ManPy

Discrete event simulation in python

  • Discrete event utilities, with a focus on manufacturing
  • Built on SimPy, NumPy, and SciPy
  • Open source, commercially usable - LGPL license

Manufacturing Objects

  • Provision of simulation components for modelling manufacturing systems.
  • Development of simulation models in simple Python Scripts
  • Customization of new components
  • Creation of domain specific object repositories

Applications

  • Workflow simulation: Simulate project dedicated manufacturing systems.
  • Job Shop simulation: Allocate tasks to constraint resources such as workers or machines.
  • Capacity Planning: Optimize utilization of constrained resources.

Interface External Software

  • KE tool: Connect simulation to organisational data.
  • DREAM GUI: Build and execute ManPy models in a drag&drop UI.
  • Interconnect simulation with scenario analysis and optimization.
  • Interface with ERP5 coming soon
