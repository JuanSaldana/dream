ManPy

Discrete event simulation in python

  • Discrete event utilities, with a focus on manufacturing
  • Built on SimPy, NumPy, and SciPy
  • Open source, commercially usable - LGPL license

Task Allocation

Allocate tasks to constraints resources such as workers or machines.

Applications: Manufacturing planning, Job Shop optimisation

Algorithms: Ant Colony Optimization

Capacity Planning

Optimize utilisation of constrained resources.

Applications: Manufacturing planning, Project optimisation

Algorithms: Ant Colony Optimization, Genetic Algorithms

Built in Objects

Comes with a set of existing ready to use objects to model industrial scenarios.

Applications: Manufacturing planning, Project optimisation
